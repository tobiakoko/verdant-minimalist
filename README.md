# Lab Website Template - Arts Edition

A website template for research labs in the arts and humanities. Based on the Sciences template with arts-specific styling.

## Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Design Differences from Sciences Template

- **Color Palette**: Violet and fuchsia tones instead of green
- **Typography**: Playfair Display (serif) for headings, Inter for body text
- **Accent Colors**: Purple gradients with amber highlights

## Configuration

This template connects to the same Sanity CMS project as other templates. Update your `.env.local` with your Sanity credentials:

```
NEXT_PUBLIC_SANITY_PROJECT_ID="your_project_id"
NEXT_PUBLIC_SANITY_DATASET="production"
```
